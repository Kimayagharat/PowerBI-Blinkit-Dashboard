# Blinkit Grocery Sales Dashboard

## 📌 Project Overview
This Power BI dashboard project analyzes Blinkit's grocery sales performance, customer ratings, outlet establishment trends, and item category performance.

The dashboard helps understand business growth, customer preferences, and outlet performance using interactive visualizations and KPIs.

# 🎯 Business Problem
Blinkit wants to analyze:
- Sales performance across outlet types
- Customer rating trends
- Best-selling product categories
- Outlet establishment growth
- Impact of outlet size and location on sales
This dashboard provides insights to support business decisions and improve sales strategy.

# 🛠️ Tools & Technologies Used
- Power BI
- DAX
- Data Cleaning
- Data Visualization
- Excel / CSV Dataset

# 📊 Dashboard Features
- Total Sales Analysis
- Average Sales
- Average Ratings
- Number of Items Sold
- Outlet Size Analysis
- Outlet Location Analysis
- Item Type Performance
- Fat Content Analysis
- Outlet Establishment Trends

# 📈 Key KPIs
| KPI | Description |
|------|-------------|
| Total Sales | Overall revenue generated |
| Average Sales | Average sales per item |
| Average Rating | Customer satisfaction rating |
| Number of Items | Total items analyzed |

# 🔍 Key Insights
- Tier 3 outlets generated higher sales compared to Tier 1 outlets.
- Regular fat products contributed more to overall sales.
- Supermarket Type 1 outlets showed the best overall performance.
- Fruits and snack items were among the highest-selling categories.
- Medium-sized outlets generated balanced sales and customer ratings.
